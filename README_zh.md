# 快速开始

这是一个基于斗罗大陆世界观的随机人生模拟小游戏。玩家可以通过转盘决定时间线、身份、武魂与成长路线，并在不同时代中经历修炼、奇遇、战斗、成神与回顾往生等内容。

## 运行方式

### 仅打开前端

适合快速体验基础玩法。

1. 打开 public/index.html。
2. 直接在浏览器中开始游戏。

说明：

- 在纯静态模式下，存档会回退到浏览器 localStorage。
- 这种方式不依赖 Node.js，但存档只保存在当前浏览器环境中。

### 启动前后端分离版本

适合完整体验存档接口与本地持久化能力。

前置条件：

- 已安装 Node.js

启动步骤：

1. 进入 server 目录。
2. 安装依赖。
3. 启动服务。
4. 在浏览器中访问 <http://localhost:3000>。

可执行命令：

```bash
cd server
npm install
npm start
```

服务说明：

- 后端基于 Express。
- 默认端口为 3000。
- public 目录会作为静态资源目录提供。
- 存档摘要保存在 server/save-list.json。
- 完整存档内容保存在 server/saves 目录。

## 项目结构

```text
public/
  index.html         前端页面入口
  css/style.css      界面样式
  js/app.js          核心玩法逻辑
  js/api.js          前后端存档通信与降级逻辑
server/
  server.js          Express 服务入口
  save-list.json     存档列表
  saves/             完整存档数据
```

## 功能介绍

## 核心玩法

- 转盘开局：通过转盘依次抽取时间线、种族身份、背景与成长条件，构建一局新的随机人生。
- 双路线体验：既可以以魂师身份开局，也可以直接以魂兽身份进入成长流程。
- 快速随机：一键同时生成魂师与魂兽候选，适合快速开局与反复刷初始。
- 武魂觉醒：完成开局抽取后进入觉醒阶段，确认武魂、先天魂力与初始资质。

## 成长与事件系统

- 年度事件转盘：每年会触发修炼、学院、伴侣、强敌、拍卖、奇遇、时间线剧情等不同事件。
- 多时代内容：包含多个斗罗系列时间线，不同时代对应不同学院、成长上限与剧情池。
- 魂环系统：可以通过猎杀、魂灵契约或特殊方式获取魂环，并生成对应技能与收益。
- 魂兽成长：魂兽路线支持年份成长、血脉相关能力以及化形等内容。
- 封号与成神：在高等级阶段可以进入封号斗罗分支，并进一步选择追求神位或进入更高层次路线。
- 社交剧情：包含伙伴、师承、宿敌、道侣等内容，角色成长不仅取决于数值，也受剧情事件影响。

## 系统功能

- 成就系统：覆盖实力、魂兽、生存、剧情、稀有与时代等多个类别。
- 往生回顾：可以查看历史存档，重新载入过去的人生。
- 本地与接口双模式存档：当后端可用时走 API 保存；后端不可用时自动回退到 localStorage。
- 健康检查接口：后端提供 /api/health 用于确认服务状态。

## 页面概览

## 1. 标题页

主界面展示游戏标题、版本信息与四个主要入口：

- 开始转世：进入完整转盘开局流程。
- 快速随机：直接生成一名魂师和一只魂兽供玩家二选一开始。
- 回忆往生：查看已有存档并继续过去的人生。
- 成就系统：进入成就殿堂查看已解锁目标。

## 2. 快速随机预览页

该页面会并排展示一个魂师候选与一个魂兽候选。玩家可以：

- 重新随机初始角色。
- 选择以魂师身份开始。
- 选择以魂兽身份开始。

## 3. 转盘页

转盘页负责完成开局抽取，是整个流程的核心入口。页面会展示：

- 当前是第几次抽取。
- 本轮抽取的主题，例如时间线或身份。
- 转盘结果与对应说明。
- 下一步按钮，用于逐步推进开局流程。

## 4. 武魂觉醒页

在这里会确认角色最终的开局配置，包括武魂、先天魂力与部分核心资质。确认后即可正式进入人生模拟阶段。

## 5. 人生模拟页

这是主要游玩界面，通常由两部分组成：

- 左侧信息区：展示年龄、魂力、武魂、魂环、伙伴、名声等成长状态。
- 右侧事件区：展示年度事件日志、可执行操作与阶段性选择。

高阶阶段还会出现：

- 封号斗罗道路选择面板
- 成神路线选择面板
- 特殊事件弹窗

## 6. 小转盘覆盖层

部分系统不会回到主转盘，而是以覆盖层形式出现，例如：

- 猎杀魂环
- 魂灵契约
- 邂逅伴侣
- 年度事件抽取

这种设计让主流程连续，同时保留“命运转盘”的随机体验。

## 7. 成就殿堂与往生回忆

- 成就殿堂：集中展示不同类别的成长目标与已解锁结果。
- 往生回忆：展示历史存档列表，支持读取或管理过往记录。
